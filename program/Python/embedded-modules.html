re与正则表达式

正则表达式（Regular Expression，简称Regex）描述了一种字符串匹配的模式（Pattern），可以用来检查一个字符串是否含有某种子字符串。构造正则表达式的方法和创建数学表达式的方法一样，即用多种元字符与运算符可以将小的表达式结合在一起来创建更大的表达式。正则表达式的组件可以是单个的字符、字符集合、字符范围、字符间的选择或者所有这些组件的任意组合。

0 注意

  1. ==re.search()和re.match()的区别是：re.match()只匹配字符串的开始，如果字符串的开始不符合正则表达式，则匹配失败，函数返回None；而re.search()匹配整个字符串，并返回第一个成功的匹配。==

  2. re.match()和re.search()只匹配一次；
    re.findall()匹配所有；
    re.sub()默认匹配所有，也可以指定匹配几次。

  3. 为避免转义产生问题，正则表达式前都推荐用raw string原生字符串，即r’…’。

  4. 很多时候，我们匹配时，只需要获取匹配结果在字符串中的位置、本身长度等信息，而re模块的常用功能并不支持，这种情况我们可以用sub中的repl函数将其参数matched对象的信息替换匹配到的内容。如下面的实例3。即结合sub函数的形参repl函数即可输出所有匹配到的match对象。

  5. match对象自带的属性和方法是针对匹配到的子串而言的，注意区分子串和原字符串的区别。

  6. 如果findall有子表达式，则直接匹配所有子表达式，但不匹配完整的表达式，如果需要匹配完整表达式，则需要将完整表达式用括号括起来。

  7. 关于findall带括号与不带括号的区别
