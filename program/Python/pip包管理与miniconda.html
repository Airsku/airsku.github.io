python venv

venv是python3自带的创建虚拟环境的工具。虚拟环境不激活也能使用，直接找到虚拟环境下的python解释器运行代码即可。

  1. 创建虚拟环境：python -m venv d:\venv\myvenv
  2. 激活虚拟环境：venv\myvenv\Scripts\activate
  3. 退出虚拟环境：deactivate
  4. 安装requirements：pip install -r requirement.txt
  5. 生成requirements：pip freeze > requirements.txt
  6. 删除虚拟环境：直接删除文件夹
